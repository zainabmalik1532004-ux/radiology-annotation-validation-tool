# Radiology Image Annotation & Quality Validation Tool

## Overview
A Python/Streamlit-based tool for medical image annotation and dataset 
labeling, built to support AI model training workflows in radiology.

## Problem it solves
Preparing high-quality annotated medical imaging datasets is one of the 
biggest bottlenecks in radiology AI development. This tool standardizes 
the annotation process and validates data quality before it reaches the 
AI pipeline.

## Features
- Medical image annotation with DICOM-compatible dataset structures
- Real-time validation dashboard flagging annotation errors
- Annotation consistency scoring across imaging samples
- Structured export of data schemas and validation criteria

## Tech Stack
- Python
- Streamlit
- DICOM
- Label Studio
- MySQL
